# NetPractice

## Description
NetPractice is a project that aims to introduce the basics of networking through 10 interactive levels. The project offers practical experience in network administration with each level increasing in difficulty with network diagrams which need configuring and troubleshooting.

## Instructions
1. Download and extract net_practice.1.7.tgz
2. Run index.html in your application of choice.
3. 
  - Choose the "Training" interface for going up the levels starting from 1.
  - Choose the "Evaluation" interface for 3 random levels between 7-10 to practice on.
4. In each level, press the "Get my config" to download a .json file which holds the contents of the different fields in the network configuration.
5. Submission requirements for the project are completing all 10 levels, submitting said levels' configs into the given school repository along with this README.md and completing the peer evaluations.
